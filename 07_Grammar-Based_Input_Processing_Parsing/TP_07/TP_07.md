## Deskripsi

program ini dirancang untuk mengonversi data masukan yang variatif menjadi sebuah lari (array) angka yang bersih dan valid. Proses diawali dengan tahap normalisasi, di mana fungsi secara cerdas mendeteksi apakah input berupa array atau string; jika berupa string, kode melakukan tokenisasi menggunakan ekspresi reguler (regex) untuk memecah teks berdasarkan pemisah koma atau spasi. Setelah data terpecah menjadi elemen-elemen individu, kode menerapkan prinsip pembersihan data melalui metode trimming untuk menghapus spasi liar dan melakukan filter ketat terhadap nilai non-numerik. Hasil akhirnya adalah sebuah struktur data yang konsisten, hanya berisi angka murni, dan aman dari error akibat karakter sampah atau format teks yang tidak rapi.
