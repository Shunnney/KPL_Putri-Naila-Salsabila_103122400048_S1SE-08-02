## Deskripsi

Program ini berfungsi untuk mem-parsing (membaca dan mengubah) isi file robots.txt menjadi struktur data JavaScript yang terorganisir. Program bekerja dengan membaca teks baris per baris, lalu mengidentifikasi instruksi seperti User-agent, Allow, Disallow, dan Sitemap. Setiap User-agent disimpan sebagai key dalam objek agents, sementara aturan Allow dan Disallow disimpan dalam bentuk array untuk masing-masing agen. Selain itu, semua URL Sitemap dikumpulkan dalam array terpisah di tingkat utama. Program juga mengabaikan baris kosong, komentar, dan nilai kosong (misalnya Disallow: tanpa isi), sehingga hasil akhirnya bersih dan sesuai dengan kebutuhan pengujian. Struktur output ini memudahkan akses terhadap aturan crawling untuk setiap user-agent secara efisien.
