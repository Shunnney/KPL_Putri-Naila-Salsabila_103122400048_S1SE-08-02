## Deskripsi

Program ini mengimplementasikan fungsi hitung yang dirancang untuk memproses sebuah string secara dinamis melalui satu struktur perulangan tunggal. Dengan menerima dua parameter input berupa teks dan tipe perhitungan, fungsi ini mampu beralih mode secara cerdas contohnya mode "semua" akan mengakumulasi seluruh karakter termasuk spasi, sementara mode "huruf" akan mengabaikan spasi menggunakan perintah continue untuk hanya menghitung karakter yang tampak.
